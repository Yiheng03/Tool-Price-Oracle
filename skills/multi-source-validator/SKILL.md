# 多源现货校验器

用于对金属当日现货价做多源交叉验证，判断报价是否可直接用于 D+1 波动判断和刀具成本影响估算。

## 核心职责

1. 从多个公开来源采集同一金属的当日或最近交易日价格。
2. 统一单位、币种和报价口径。
3. 计算来源之间的差异。
4. 输出可信度：高 / 中 / 低。
5. 标注推荐采用的价格和原因。

## 推荐数据源

| 来源 | 适用金属 | 说明 |
|------|----------|------|
| SMM / 上海有色网 | 有色金属 | 国内现货参考 |
| 长江有色 | 铜、铝、铅、锌、锡、镍 | 国内现货参考 |
| 生意社 | 多品种 | 趋势和报价补充 |
| LME / SHFE | 交易所金属 | 国际/期货信号补充 |
| 行业协会/公开报价 | 钨、铁矿石等 | 专用品种补充 |

## 工作流程

### 步骤 1：确定校验对象

输入金属代码、目标日期和可用主报价。

### 步骤 2：采集多源报价

搜索并记录：

- 报价来源。
- 报价时间。
- 价格。
- 单位。
- 币种。
- 报价口径（均价、区间、中间价、主力合约等）。

### 步骤 3：统一口径

必要时做单位换算，并标注换算依据。

### 步骤 4：计算差异

```text
source_diff_pct = abs(source_price - recommended_price) / recommended_price
```

可信度规则：

| 多源差异 | 可信度 | 处理 |
|----------|--------|------|
| ≤ 1% | 高 | 可直接采用 |
| 1%~3% | 中 | 采用均值或主流来源，说明差异 |
| > 3% | 低 | 降低 D+1 判断置信度，列出报价口径差异 |

## 输出格式

```markdown
## 多源现货校验 [{metal}] [{date}]

| 来源 | 价格 | 单位 | 报价时间 | 口径 |
|------|------|------|----------|------|

**推荐采用价格：** {price} {unit}
**可信度：** 高/中/低
**说明：** {差异原因和采用依据}
```

## 使用时机

- 关键金属的 D+1 判断前。
- 当主报价源缺失或延迟时。
- 当不同来源价格差异明显时。
- 第二天核对实际价格时。
