# 下游需求感知器

不仅看供给端（矿山、政策），还要看需求端——工厂到底要不要买刀具？本 Skill 跟踪制造业需求指标，弥补"只预测成本、不预测需求"的盲区。

## 为什么需要需求感知

| 供给端会说 | 需求端不说 | 后果 |
|-----------|----------|------|
| 钴价涨 20%，刀具成本上升 | 但汽车厂减产 30%，没人买刀具 | 成本涨了但价格涨不上去 |
| 铁矿石跌了 | 但基建刺激来了，HSS 刀具需求暴增 | 成本降了但价格不应降 |
| 钨价涨 700% | 但替代刀具（陶瓷）订单激增 | 需求结构正在变化 |

**结论：只看供给侧的成本是片面的，供需两端才能判断真实价格方向。**

## 核心指标

### 指标 1：中国制造业 PMI

| 数据源 | 搜索关键词 | 更新频率 | 阈值 |
|--------|-----------|---------|------|
| 国家统计局 | `中国 官方 PMI 2026年` | 每月1日 | >50.5 扩张 / <49.5 收缩 |
| 财新 | `财新 PMI 2026年` | 每月1日 | 同上 |

**对刀具需求的影响：**
- PMI >50.5 持续 2 个月 → 制造业扩张 → 刀具需求 ↑
- PMI <49.5 持续 2 个月 → 制造业收缩 → 刀具需求 ↓
- PMI 在 49.5-50.5 之间 → 中性

### 指标 2：机床产量/订单

| 数据源 | 搜索关键词 | 更新频率 |
|--------|-----------|---------|
| 中国机床工具工业协会 | `机床产量 2026 同比` | 月 |
| 日本机床工业会 | `Japan machine tool orders 2026` | 月 |
| 德国 VDW | `German machine tool orders 2026` | 季 |

**机床是刀具的领先指标**（机床买了就要配刀具）：
- 机床订单同比 >+10% → 刀具需求将在 1-2 个月内上升
- 机床订单同比 <-10% → 刀具需求疲软

### 指标 3：分行业下游景气度

| 行业 | 核心指标 | 搜索关键词 | 对刀具的影响 |
|------|---------|-----------|------------|
| 汽车 | 中国汽车产量同比 | `中国汽车产量 2026年` | 最大刀具消费行业 |
| 航空航天 | 波音/空客交付量 | `Boeing Airbus deliveries 2026` | 高价值刀具 + 钛合金刀具 |
| 模具 | 中国模具出口额 | `模具出口 2026` | 硬质合金刀具需求 |
| 通用机械 | 固定资产投资 | `通用设备制造业 投资 2026` | 基础需求 |
| 电子 | 3C 产品出货量 | `手机出货量 2026` | 微小径刀具 |
| 新能源 | 锂电池/光伏产能 | `锂电 产能 2026` | 特殊材料加工刀具 |

### 指标 4：刀具行业自身信号

| 指标 | 搜索关键词 | 含义 |
|------|-----------|------|
| 刀具龙头企业营收 | `山特维克 营收 2026` `肯纳金属 财报 2026` | 行业方向标 |
| 刀具进出口数据 | `刀具 进口 2026` `刀具 出口 2026` | 中国刀具贸易平衡 |
| 刀具行业景气指数 | `刀具行业 景气 2026` | 综合指标 |

## 工作流程

### 步骤 1：收集核心需求指标

使用 `WebSearch` 获取最近的：
1. 中国 PMI（官方 + 财新）
2. 机床产量/订单数据
3. 主要下游行业最新动态
4. 可选：刀具龙头财报

### 步骤 2：计算需求压力指数

```
需求压力指数 (DDI, Demand-Driven Index) = 

  0.35 × (PMI_signal)           # PMI: <49.5=-1, 49.5-50.5=0, >50.5=+1
+ 0.30 × (machine_tool_signal)   # 机床订单: <-10%=-1, -10~+10%=0, >+10%=+1
+ 0.20 × (auto_demand_signal)    # 汽车产量同比: <0%=-1, 0~5%=0, >5%=+1
+ 0.15 × (other_demand_signal)   # 综合其他行业: 多数扩张=+1, 多数收缩=-1
```

DDI 区间：
- **>0.5** → 需求旺盛 → 价格有上行压力（demand-pull）
- **-0.5 ~ 0.5** → 需求中性
- **<-0.5** → 需求疲软 → 价格有下行压力（demand-destruction）

### 步骤 3：输出需求感知简报

```
## 📊 下游需求感知 [日期]

### 核心指标

| 指标 | 当前值 | 上月值 | 同比 | 信号 |
|------|--------|--------|------|------|
| 官方 PMI | {50.2} | {49.8} | — | ↑ 回升 |
| 财新 PMI | {50.5} | {50.0} | — | ↑ 略升 |
| 机床产量同比 | {+8%} | {+5%} | {+3pp} | ↑ 加速 |
| 汽车产量同比 | {+6%} | {+4%} | {+2pp} | ↑ 增长 |
| 刀具企业营收 | {+12%} | {+8%} | {+4pp} | ↑ 景气 |

### 需求压力指数 (DDI)

**当前 DDI：{+0.6} → 🟢 需求旺盛**

### 与供给端的交叉分析

| 金属 | 供给端方向 | 需求端方向 | 综合判断 |
|------|----------|----------|---------|
| 钴 | ↑ 偏紧（刚果配额） | → 中性 | 供给驱动上行 |
| 钨 | ↑↑ 极紧（出口管制） | → 中性 | 供给断崖式上涨，与需求无关 |
| 镍 | ↑ 偏紧（印尼减产） | → 中性 | 供给驱动 |
| 铁矿石 | ↓ 偏松（西芒杜） | ↑ 旺盛（基建） | 供需博弈，方向不明确 |

### 需求端关键判断

{一句话总结：需求端对刀具价格是推升、压制还是中性？}

{一句话建议：在当前需求态势下，采购策略应如何调整？}
```

## 限制说明

- DDI 是定性指标，用于补充现货、新闻和供应链判断
- 需求数据通常有 1-2 个月的滞后，反映的是"已经发生"的需求
- 对前瞻需求的判断依赖行业经验和趋势外推
